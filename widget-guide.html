How to Use the Crypto Exchange Widget

Step 1: Access the Widget

Navigate to the exchange widget section in the Paca Finance UI

Step 2: Select Currencies

Choose the cryptocurrency you want to exchange from and to:

  • Select the source currency from the "From" dropdown
  • Select the target currency from the "To" dropdown

Step 3: Enter Amount

Input the amount you want to exchange in either the "From" or "To" field

The widget will automatically calculate the equivalent amount in the other currency

Step 4: Review Exchange Rate

Check the current exchange rate and fees displayed by the widget

Rates are updated in real-time to ensure you get the best available price

Step 5: Connect Wallet

If not already connected, click "Connect Wallet" and select your wallet provider

Step 6: Confirm Exchange

Review all details and click "Exchange" to proceed with the transaction

Important:

Always verify the transaction details in your wallet before confirming
